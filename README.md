# 🌍 TripBack

TripBack är en webbaserad app som låter dig ladda upp foton med GPS-data och visualisera dina resor på en karta. Appen använder EXIF-data, Leaflet, Mapbox, Firebase och OpenStreetMap för att skapa en interaktiv resehistorik.

## 🚀 Funktioner

- Ladda upp bilder med GPS-data
- Visa resor på interaktiv karta
- Spara resor i Firebase Firestore
- Automatisk platsidentifiering via reverse geocoding
- Responsiv design
- Offline-stöd via service worker

## 🛠 Teknologier

- HTML, CSS, JavaScript (ES Modules)
- Leaflet + Mapbox
- EXIF.js
- Firebase Firestore
- OpenStreetMap
- Service Worker (PWA)
